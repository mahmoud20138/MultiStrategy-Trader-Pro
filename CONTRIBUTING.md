# Contributing to Trading System Pro

Thank you for your interest in contributing to Trading System Pro!

## How to Contribute

### Reporting Bugs

1. **Check existing issues** - Search the issue tracker to see if your bug has been reported
2. **Create a detailed issue** - Include:
   - Clear title describing the problem
   - Steps to reproduce the bug
   - Expected vs actual behavior
   - Your environment (Python version, MT5 terminal version, OS)
   - Any relevant logs or screenshots

### Suggesting Features

1. Open an issue with the label `feature-request`
2. Describe the feature you'd like to see
3. Explain why this feature would be valuable
4. Include any relevant research or references

### Pull Requests

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature`
3. **Make your changes** following the existing code style
4. **Add tests** if applicable
5. **Update documentation** if your change affects the API or usage
6. **Submit a PR** with a clear description of what you fixed or added

## Code Style

- Use type hints where possible
- Follow PEP 8 guidelines
- Add docstrings to new functions and classes
- Keep lines under 100 characters when practical

## Testing

Run tests before submitting a PR:
```bash
python -m pytest tests/
```

## Questions?

Open an issue with the `question` label if you need help or have questions about contributing.
